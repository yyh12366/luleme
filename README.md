# 每日打卡应用 ✨

一个极简的打卡应用，专注核心功能：打卡和日历。

## 🌟 功能特性

### 📅 打卡功能
- **一天可以打卡多次**：不限制打卡次数
- 一键打卡，简单直接
- 精美的庆祝动画
- 显示今日已打卡次数
- 自动计算连续打卡天数

### 📆 日历视图
- 直观的月历展示
- 已打卡日期高亮显示
- **显示每天的打卡次数**：多次打卡会在日期右上角显示数字
- 支持切换月份查看历史记录
- 今日日期特殊标记

### 💡 激励语录
- 10条精选励志名言
- 自动切换显示

## 🚀 使用方法

### 1. 打开应用
直接在浏览器中打开 `index.html` 文件即可使用。

### 2. 开始打卡
点击"立即打卡"按钮完成打卡。

- ✅ 一天可以打卡多次
- ✅ 每次打卡都会记录精确时间
- ✅ 打卡成功后会显示庆祝动画
- ✅ 状态栏会显示今日已打卡次数

### 3. 查看历史
使用日历上方的左右箭头按钮，可以切换月份查看历史打卡记录。

## 💾 数据存储

应用使用浏览器的 LocalStorage 存储数据：
- 所有打卡记录（包含时间戳和日期）
- 连续打卡天数

**注意**：
- 数据仅存储在本地浏览器中
- 清除浏览器数据会导致记录丢失
