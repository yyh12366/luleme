# 快速开始指南

## 🚀 5分钟上手

### 第一步：打开应用
双击 `index.html` 文件，或在浏览器中打开它。

### 第二步：开始打卡
点击大大的"立即打卡"按钮 ✓

**你会看到：**
- 🎆 精美的庆祝动画
- 📝 今日打卡记录中出现新的打卡时间
- 📊 统计数据自动更新
- 📅 日历上今天的日期被高亮

### 第三步：多次打卡
想打几次就打几次！每次都会：
- 记录精确时间
- 触发庆祝动画
- 更新打卡次数
- 添加到今日记录

### 第四步：添加目标
1. 点击"我的目标"区域的"+ 添加"按钮
2. 输入目标名称（如：每天运动30分钟）
3. 选择一个喜欢的图标
4. 点击"保存"

**目标会自动追踪：**
- 每次打卡都会更新目标进度
- 显示本月完成百分比
- 可视化进度条

### 第五步：查看历史
- 📅 **日历视图**：点击左右箭头切换月份
- 📝 **今日记录**：查看今天所有打卡时间
- 📊 **统计数据**：查看总打卡、本月打卡、完成率

## 💡 使用技巧

### 1️⃣ 番茄工作法
每完成一个25分钟的专注时段就打一次卡
- 上午：打卡4次 = 2小时专注
- 下午：打卡4次 = 2小时专注
- 晚上：打卡2次 = 1小时专注

### 2️⃣ 健身记录
- 早上：晨跑打卡 🏃
- 中午：午休瑜伽打卡 🧘
- 晚上：力量训练打卡 💪

### 3️⃣ 学习追踪
- 每学习30分钟打一次卡
- 每完成一个章节打一次卡
- 每做完一套题打一次卡

### 4️⃣ 喝水提醒
- 每喝一杯水就打卡
- 目标：一天8杯水 = 8次打卡

### 5️⃣ 写作激励
- 每写500字打一次卡
- 每完成一个段落打一次卡
- 保持写作节奏

## 📱 手机使用

### 添加到主屏幕（iOS）
1. 在 Safari 中打开应用
2. 点击分享按钮
3. 选择"添加到主屏幕"
4. 像原生应用一样使用！

### 添加到主屏幕（Android）
1. 在 Chrome 中打开应用
2. 点击菜单（三个点）
3. 选择"添加到主屏幕"
4. 完成！

## 🎯 最佳实践

### ✅ 推荐做法
- 设定明确的打卡规则（如：每30分钟一次）
- 定期查看统计数据，了解自己的习惯
- 设置合理的目标，循序渐进
- 保持连续打卡，建立习惯

### ❌ 避免
- 无意义的频繁打卡（失去记录意义）
- 忘记打卡后补打（保持真实性）
- 设置过多目标（专注更重要）

## 📞 需要帮助？

### 常见问题

**Q: 数据会丢失吗？**
A: 数据存储在浏览器本地，除非清除浏览器数据，否则不会丢失。

**Q: 可以导出数据吗？**
A: 当前版本暂不支持，但数据存储在 LocalStorage 中，可以通过浏览器开发者工具查看。

**Q: 可以同步到多个设备吗？**
A: 当前版本是纯本地应用，不支持云同步。

**Q: 一天打卡多少次合适？**
A: 根据你的目标决定！建议3-10次之间，既有记录价值又不会太频繁。

**Q: 连续打卡断了怎么办？**
A: 不要气馁！重新开始，关键是养成习惯。
